# Docx Tracked Edits - 核心语法

## 模板头部

```yaml
---
author: 姓名
source: 路径.docx
output: 路径_修订.docx
---
```

## 编辑类型

| 类型 | 语法 |
|------|------|
| **替换** | `将 "old" 改为 "new"` |
| **开头插入** | `在开头插入: text` |
| **末尾插入** | `在末尾插入: text` |
| **删除** | `删除: "text"` 或 `删除: "text" (第15-18字符)` |
| **格式** | `居中对齐, 加粗, 行距1.5倍` |
| **表格** | `第N行下方加一行`, `删掉第N行`, `合并第N行的第X-Y列` |
| **样式** | `Normal 样式: 字号10pt, 加粗` |
| **全局** | `将 "old" 改为 "new"`（无 Para 前缀） |

## 批注格式

```
Para {N}: {标题}
{批注内容}
> 选中范围: 第{start}-{end}字符
> 缩写: {缩写}
```

## 规则

1. **极简原则**：只替换词组，不要替换整句
2. **位置信息**：文本多次出现时添加 `(第N-M字符)`
3. **工具多样化**：使用删除/插入，不要只用替换

完整示例和错误处理见 REFERENCE-zh.md。
